## Controls

| Key            | Action          |
|----------------|-----------------|
| Arrow Up / W   | Move paddle up  |
| Arrow Down / S | Move paddle down|
| Space          | Start / Restart |

## Rules

- First to **7 points** wins.
- The ball speeds up with each paddle hit.
- The CPU opponent tracks the ball with limited reaction speed.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Build output is written to `dist/`.

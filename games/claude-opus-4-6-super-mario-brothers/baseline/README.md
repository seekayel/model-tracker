# Super Mario Brothers

A browser-based platformer clone inspired by Super Mario Brothers, built with vanilla JavaScript and Vite.

## Controls

| Action | Keys |
|--------|------|
| Move Left | Arrow Left / A |
| Move Right | Arrow Right / D |
| Jump | Arrow Up / W / Space |
| Duck (Big Mario) | Arrow Down / S |
| Run / Fireball | Shift / X |
| Start / Restart | Enter |

## Features

- Side-scrolling platformer with procedurally generated pixel-art sprites
- Mario can grow big (mushroom), shoot fireballs (fire flower), and gain invincibility (star)
- Enemies: Goombas and Koopa Troopas (with kickable shells)
- Breakable bricks and question blocks with items
- Pipes, staircases, and a flagpole finish
- Score, coins, lives, and timer HUD
- Sound effects via Web Audio API
- Title screen and game over / win flow

## Run / Build

```bash
npm install
npm run dev       # Start dev server
npm run build     # Build to dist/
npm run preview   # Preview production build
```

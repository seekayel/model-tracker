# Pong Game

A simple, browser-based Pong game clone built with TypeScript and Vite.

## Controls

-   **Player 1 (Left Paddle):**
    -   `W`: Move Up
    -   `S`: Move Down
-   **Player 2 (Right Paddle):**
    -   `ArrowUp`: Move Up
    -   `ArrowDown`: Move Down
-   **Game:**
    -   `Enter`: Start or Restart the game

## Development

To run the game in development mode:

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Start the dev server:
    ```bash
    npm run dev
    ```

## Build

To build the static assets for production:

1.  Build the project:
    ```bash
    npm run build
    ```
2.  The static files will be generated in the `dist/` directory.

## Preview

To preview the production build locally:

1.  Run the preview server:
    ```bash
    npm run preview
    ```

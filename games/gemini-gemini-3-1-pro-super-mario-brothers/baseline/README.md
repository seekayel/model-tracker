# Super Jumper

A simple browser-based game inspired by Super Mario Brothers.

## Development

To run the game in a local development environment, follow these steps:

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Start the development server:**
    ```bash
    npm run dev
    ```

    This will open the game in your browser.

## Build

To build the static assets for production, run:

```bash
npm run build
```

The output will be in the `dist/` directory. You can preview the production build with:

```bash
npm run preview
```

## Controls

-   **Left Arrow:** Move left
-   **Right Arrow:** Move right
-   **Up Arrow:** Jump

## Gameplay

-   Jump on platforms.
-   Avoid falling into pits.
-   The game ends if you fall off the bottom of the screen.
-   Press "Restart" to play again.

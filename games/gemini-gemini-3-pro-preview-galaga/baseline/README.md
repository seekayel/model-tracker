# Galaga Clone

A browser-based arcade shooter inspired by Galaga. Built with Vanilla JS, HTML5 Canvas, and Vite.

## Controls
- **Move Left/Right**: Arrow Keys or A/D
- **Shoot**: Spacebar

## Setup and Build
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Build for production (outputs static files to `dist/`):
   ```bash
   npm run build
   ```

# Gemini Pong

A simple, browser-based Pong clone built with Vanilla JS and Vite.

## Controls

-   **Start Game / Restart**: Press `SPACE`
-   **Move Paddle**: `W` / `S` or `Arrow Up` / `Arrow Down`

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```
    The output will be in the `dist/` directory.

4.  Preview production build:
    ```bash
    npm run preview
    ```

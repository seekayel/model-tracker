# Pong Clone

A browser-based, client-side Pong clone built with JavaScript and the HTML5 Canvas API.

## Requirements
- Node.js

## Setup and Running

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
   The built static files will be placed in the `dist/` directory. You can preview the build with `npm run preview`.

## Controls
- **Player 1 (Left Paddle)**: `W` and `S` keys, or `Up Arrow` and `Down Arrow`.
- **Start / Restart Game**: `Spacebar` or `Enter`.

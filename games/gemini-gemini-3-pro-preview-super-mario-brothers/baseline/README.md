# Super Mario Clone

A simple browser-based platformer clone inspired by Super Mario Brothers.
Built with Vanilla JS, HTML5 Canvas, and Vite.

## Controls
- **Left/Right Arrows**: Move left/right
- **Up Arrow / Space**: Jump
- **Enter**: Start/Restart game

## How to play
Jump on the brown enemies (Goombas) to defeat them and earn points.
Avoid touching them from the sides.
Reach the yellow flag pole at the end of the level to win!
Watch out for gaps in the floor.

## Build and Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production (static assets):
   ```bash
   npm run build
   ```
   The output will be in the `dist` directory.

4. Preview production build:
   ```bash
   npm run preview
   ```
